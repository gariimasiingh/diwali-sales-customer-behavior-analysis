# 🪔 Customer Behavior Analysis – Diwali Sales

## 📌 Project Overview
This project analyzes customer purchasing behavior during Diwali sales using Python and Power BI.

## 📊 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## 🔍 Key Insights
- Female customers contribute more to total sales
- Age group 26–35 drives maximum revenue
- Top states: UP, Maharashtra, Karnataka
- Food, Clothing, and Electronics are top categories
- Married women aged 26–35 show strong purchasing behavior
